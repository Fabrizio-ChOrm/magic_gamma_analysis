# MAGIC Gamma Telescope Analysis

## Descripción
Proyecto de análisis descriptivo del dataset MAGIC Gamma Telescope de UCI.

## Objetivo
Analizar las características de eventos gamma y eventos hadrónicos utilizando técnicas de exploración y visualización de datos.

## Dataset
Magic Gamma Telescope Dataset (UCI Repository):
https://archive.ics.uci.edu/dataset/159/magic+gamma+telescope

## Variables analizadas
- fLength
- fWidth
- fSize
- fConc
- fConc1
- fAsym
- fM3Long
- fM3Trans
- fAlpha
- fDist

## Herramientas utilizadas
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn
- Github
- Overleaf

# Estructura del proyecto
- data/: archivos de datos
- notebooks/: notebooks del análisis de los datos
- images/: gráficos y visualizaciones
- poster/: póster científico
- paper/: informe académico
